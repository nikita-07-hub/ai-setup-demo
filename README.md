# My AI Setup Task

## What tools I installed
- Cursor IDE (from https://cursor.com)
- Claude Code extension (added in Extensions → search "Claude Code")
- Codex extension (added in Extensions → search "Codex")

## What steps I completed
1. Installed Cursor IDE on my laptop.
2. Added the Claude Code extension.
3. Added the Codex extension.
4. Created a GitHub account at https://github.com.
5. Created a new public repository called "ai-setup-demo".
6. Installed the Command Line Developer Tools after getting a git-related system prompt.
7. Connected the repository to Cursor.
8. Created this `README.md` file and added the task details.
9. Committed and pushed the changes to GitHub.

## Issues I ran into and how I solved them
  - Cursor’s interface was new to me, so it took some time to understand where different commands and options were located.
  - I could not find the "Clone Directory" option directly at first, so I used the keyboard command search to find the correct option and continue.
  - While trying to connect the repository, I got a message saying that the `git` command required the Command Line Developer Tools.
  - At first, I paused the installation because the estimated time was extremely long. It initially showed around 72 hours and after running for a long time, the estimate kept changing to 1 day or 2 days, which seemed like a bug or system issue.
  - After that, I restarted Cursor and tried the installation process again. The Command Line Developer Tools were then installed successfully in under 5 minutes.
  - After installing git tools and cloning the repository properly, I recreated `README.md` inside the cloned folder in Cursor.
